What is difference between isA and hasA relationship?

  • A car has four wheels (hasA relationship)
    - wheel is a member of a car
  • A butterfly is a type of insect (isA relationship)
    - suggests inheritance
    - insect is “super-class” and butterfly is “sub-class”

Why static attributes and methods?

  • A class needs to know how many run-time objects are available at any point in time. A static integer counter object is made available in the class scope, so it is neither file scope or global scope.
